What is Polar?

Polar is a web application that allows speakers to ask questions to their audience and get answers in real time.

Create

With Polar, you can create questions for your audience

Take part

As an audience member, you can answer questions

Observe

As an audience member, you can observe the submitted results in realtime without participating

Where do I start?

The project is on GitHub and is not ready to use but you can try the interface right now
